La base de données étant en local, vous trouverez le script de la base à insérer dans Wamp, le mot de passe peut être 
modifier dans le fichier App.config.

Le nom de la base de données à créer est ynovprojet.
Le server est localhost.
Le username est root et il ne contient pas de mot de passe.

Si ces votre configuration est différentes, merci de changer ces info dans le fichier App.config ligne 80.

Sinon un mazeprojetynov.exe vous permettera de tester la création de map et l'algorithme A* sans les parties en liens
avec la base de données.
